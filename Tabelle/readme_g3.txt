Alle Flughäfen werden in der csv Datei airports_overview.csv aufgelistet. Das Modell sagt ledigtlich Flüge basierend auf der ID vorher. Es braucht zur Auswahl einerseits den Ankunftsflughafen und den Abflugsflughafen.


Wenn es noch etwas realistischer sein sollte dann kann aus der Tabelle flights_data.csv aus dem gegebenen Datensatz eine Fluglinie gewählt werden mit einer realistischen Zeit und
den zugehörigen Orten. Das müsste dann runter gefiltert werden
